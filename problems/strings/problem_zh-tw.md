**字串**就是被引號包裹起來的任意的值。

單引號或雙引號效果是一樣的：

```js
'this is a string'

"this is also a string"
```
# 注

為了保持一致的風格，本教程中我們將只使用單引號。

## 挑戰：

創建一個名為 `strings.js` 的檔案。

在檔案中像這樣創建一個名為 `someString` 的變數：

```js
var someString = 'this is a string';
```

使用 `console.log` 印出變數 **someString** 到終端。

執行下面的命令來檢查你寫的程式是否正確：

`javascripting verify strings.js`
